# Projeto 4: Triple Espresso (Cafeteria Triple Peaks)

## Descrição do Projeto
Esta é uma landing page desenvolvida como projeto final do Sprint 4 do curso de Engenharia de Software da TripleTen. O site representa o "Triple Espresso", uma cafeteria aconchegante localizada na biblioteca Triple Peaks, pensada como uma zona de estudos. 

O foco deste projeto foi a aplicação de conceitos avançados de marcação e estilização, transformando um design estático em uma página web funcional e estruturada. As principais funcionalidades e tecnologias utilizadas incluem:
* **HTML5 & CSS3:** Semântica avançada e estilização rigorosa baseada em design.
* **Metodologia BEM (Block, Element, Modifier):** Refatoração da arquitetura CSS para a estrutura BEM Flat, isolando blocos em arquivos separados (`blocks/`) para um código escalável e fácil de manter.
* **Flexbox & Box Model:** Alinhamento preciso de elementos, controle de margens/preenchimentos e contorno de colapsos de margem.
* **Posicionamento (Absolute/Relative & Z-index):** Criação de composições visuais complexas, como o círculo de destaque sobreposto e posicionado atrás da logo no rodapé.
* **Multimídia e Formulários:** Incorporação de iframes do YouTube e construção de um formulário de reservas responsivo e estilizado.

## Planos de Melhoria do Projeto
Como um profissional com olhar voltado para a excelência em UI/UX, visualizo as seguintes melhorias futuras para aprimorar este produto digital:
1.  **Responsividade Total:** Implementar Media Queries para adaptar perfeitamente o layout (especialmente os blocos de receita e o formulário) a dispositivos móveis e tablets.
2.  **Micro-interações de UI:** Adicionar transições suaves (`transition`) nos botões e links além da mudança de opacidade, trazendo mais fluidez à navegação.
3.  **Dark Mode:** Criar um tema escuro alternativo, focando na acessibilidade e no conforto visual para usuários que acessam a página para estudar durante a noite.
